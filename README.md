# Mokausi anglų — Lithuanian → English learning PWA

A small, offline-first Progressive Web App that helps two Lithuanian-speaking
adults practise English every day. Phrase-level spaced-repetition flashcards,
every English phrase read aloud, leveled units that unlock as you progress, a
daily streak — all in plain HTML/CSS/JS with no backend, no accounts, no cost.

This is the daily-habit companion to the two Claude Project AI tutors
(see `CLAUDE.md`).

## Features

- **Spaced repetition** (SM-2 based) with three simple ratings: 🔁 again / 🙂 good / 😎 easy.
- **Exercises that adapt to the level** — a card starts as tap-to-reveal, then (as it's learned) becomes listening multiple-choice → typing → speaking. Each is toggleable per learner in Settings:
  - 👂 *Klausymas* — hear the English, pick the Lithuanian (offline).
  - ⌨️ *Rašymas* — type the English (offline). On by default for Mom.
  - 🎤 *Kalbėjimas* — say the English aloud, checked by speech recognition. **Needs internet** and is off by default (can misfire on accents — there's a "said it right" override).
- **Text-to-speech** — the English side is spoken automatically (Web Speech API), with a 🔊 replay button.
- **Two learner tracks** — *Tėtis* (absolute beginner) and *Mama* (a step higher). Pick once per phone; each keeps independent progress.
- **Leveled curriculum** — themed units (greetings, shopping, family, medical, numbers, directions…) that unlock as you master the previous one.
- **Add your own cards** — Settings → *Pridėti korteles*. Paste `lietuviškai = angliškai` lines (great for phrase lists generated by the Claude tutors).
- **Streak & progress** — daily streak, cards reviewed today, total learned.
- **Works offline** — service worker caches everything; fully usable in airplane mode once installed.

## Run locally

No build step. Serve the folder over HTTP (a service worker needs http/https, not `file://`). Any static server works, e.g. with VS Code's "Live Server" extension, or any tool you have on hand.

Then open the served URL in Chrome.

## Deploy to GitHub Pages (free HTTPS)

A PWA can only be *installed* and run offline over HTTPS. GitHub Pages gives that for free:

1. Create a GitHub repo and push this folder to it (the app lives at the repo root).
2. Repo → **Settings → Pages**.
3. Under **Build and deployment → Source**, choose **Deploy from a branch**.
4. Pick branch `main` and folder `/ (root)`, then **Save**.
5. After a minute, your app is live at `https://<your-username>.github.io/<repo-name>/`.

Share that one link with both phones.

## Install on an Android phone

1. Open the GitHub Pages link in **Chrome**.
2. Tap the **⋮** menu → **Add to Home screen** (or accept the install prompt).
3. Launch it from the home screen — it opens full-screen like an app.
4. First launch: tap **Mama** or **Tėtis**. That choice is remembered on this phone
   (change it later in **⚙️ Nustatymai → Keisti mokinį**).

Install separately on each phone so each parent gets their own profile and progress.

## Updating the content / app

- Edit the curriculum in [`js/decks.js`](js/decks.js) (each card is `['lietuviškai', 'English']`).
- After changing any file, bump the `CACHE` version in [`service-worker.js`](service-worker.js)
  (e.g. `v1` → `v2`) so phones pick up the new version.
- Re-deploy by pushing to GitHub.
